WebDev Notes

Here I'm collecting notes of what I've learned to reference in the future when needed. Here are some terms I've learned so far.

Nav
Nav is a tag used to create a navigation element, letting users move from one piece of content (page, section, etc) to another.
Form
Form is a tag that's used to create an interactive element that allows users to submit information to the website for collection.
Button
Button is a tag used to create an interactive element that triggers some secondary action.
section*3>h2+p*3>
This is either a tag or a shortcut that is used to create multiple "section" elements at the same time.
Div
Div is a tag that creates an invisible element used like an organizational box to house other information or elements. It doesn't have a visible effect unless styled with CSS.
